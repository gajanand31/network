Connect to Firebase to sync your contacts across all devices in real time. Takes about 5 minutes to set up.

1
Create a free Firebase project
Go to console.firebase.google.com
Click Add project, name it "My Network"
Disable Google Analytics (optional) and click Create
2
Create a Realtime Database
In your project sidebar: Build > Realtime Database
Click Create database, choose any region
Start in Test mode (you can secure it later)
Copy your Database URL (ends with .firebaseio.com)
3
Get your config
Go to Project Settings (gear icon)
Scroll to Your apps, click the Web icon (</>)
Register the app, copy the firebaseConfig object
Use the same key on all your devices. Anyone with this key can access your data.
✅
Syncing!
Your data is live on Firebase.
Open this app on any device and enter the same sync key.
